This driver is suitable for: 
Ubit WIE9260/AX200/AX210, etc. 

This driver supports:
Win10_64 bit

The minimum system requirement for this model is Microsoft Windows 10.
Previous versions of Windows 10 are not supported.

WiFi_22.0.0_Driver64_Win10 applies to Microsoft Windows 10 64-bit system (WiFi driver)
BT_22.00.0_64_Win10 applies to Microsoft Windows 10 64-bit system (Bluetooth driver)
